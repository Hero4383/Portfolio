Step 2: Create CLAUDE.md (Global Rules)

This file teaches your AI agent how to behave professionally and consistently.

# AI Agent Development Rules

## Code Quality Standards
- Follow PEP8 for Python code style
- All functions must have docstrings
- Include type hints for function parameters
- Write unit tests for all new functions

## Decision-Making Process
- Always use the 'think' tool before making complex decisions
- Check the research/ directory before using new APIs
- Verify code against knowledge graph before implementation
- Run tests after any code changes

## File Organization
- Keep related functionality in the same module
- Use clear, descriptive variable names
- Group imports at the top of files
- Include README.md for each major component

## Error Handling
- Implement try-catch blocks for external API calls
- Log errors with context for debugging
- Provide user-friendly error messages
- Always check return values before using them
